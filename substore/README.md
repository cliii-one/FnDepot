# SubStore for fnOS

> 基于 [Sub-Store](https://github.com/sub-store-org/Sub-Store) 的飞牛 fnOS 原生应用

## 功能特性

- 🔄 **订阅转换** — 支持 Clash / Surge / Loon / QX / Stash / Shadowrocket / sing-box 等多种格式
- 📝 **订阅格式化** — 过滤、排序、重命名、脚本操作
- 📦 **组合订阅** — 将多个订阅合并为一个 URL
- ☁️ **Gist 同步** — 云端备份与恢复
- 🌐 **Web 管理面板** — 基于 Vue 3 的 PWA 界面
